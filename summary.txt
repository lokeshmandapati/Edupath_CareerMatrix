EduPath – One Stop Personalized Career & Education Advisor
(Also known as SmartCareer / newsmartcareer)

OVERVIEW:
EduPath is an intelligent, full-stack career guidance and education platform designed to help students make data-driven decisions about their academic and professional futures. Using an AI-driven engine and a rich interactive interface, the application analyzes a user's skills, academic performance, and personal interests to suggest the most optimal educational streams, college paths, and career roles.

TECH STACK:
- Frontend: React.js (Vite), Tailwind CSS (for modern UI), Framer Motion (for animations), Recharts (for data visualization)
- Backend: Java Spring Boot (REST APIs, Security, Data processing)
- Database: PostgreSQL (User profiles, assessment history)
- AI Engine: Google Gemini API (Integrated for intelligent Chatbot and Roadmap generation)

CORE FEATURES & FUNCTIONALITIES:

1. Class 10 Assessment (Stream Selection)
   - Evaluates the user's marks and core interests.
   - Recommends the best academic stream (Science (PCM/PCB), Commerce, Arts/Humanities).
   - Provides recommended subjects, high-level exam paths, and actionable next steps.

2. After 12th Assessment (Degree & College Mapping)
   - Evaluates preferred work types, subjects, and location flexibility.
   - Suggests specific undergraduate courses (e.g., Engineering, Medicine, CA, Design).
   - Dynamically fetches top college recommendations tailored to the user's category (General, OBC, SC/ST) and location preferences, showing approximate cutoffs and official website links.

3. Professional Career Prediction (CFPA Engine)
   - Analyzes technical skills, project experience, certifications, and CGPA.
   - Calculates a precise Career Fit Prediction Algorithm (CFPA) score.
   - Provides an interactive "Blueprint" dashboard with comprehensive domain rankings, skill/interest alignment charts, and cross-branch opportunities.

4. AI-Powered Chatbot
   - Powered by the Google Gemini API.
   - Provides personalized 24/7 career advice by referencing the user's specific assessment history.

5. Dynamic Roadmap Generation
   - Generates step-by-step, actionable roadmaps tailored to the user's target career (e.g., "How to become a Data Scientist").
   - Adapts to the user's current educational level (Class 10, After 12th, or College Graduate).

6. Centralized Dashboard & Analytics
   - "View Insights" panel seamlessly tracks and displays the user's latest assessment results.
   - Unified interface ensuring users can easily navigate between taking tests, generating roadmaps, and talking to the AI.

7. Secure User Profiles
   - Complete authentication flow (Signup, Login) secured with JWT.
   - PostgreSQL database stores all user assessment histories securely for future reference.

VALUE PROPOSITION:
EduPath removes the guesswork from career planning. By transitioning from generic advice to hyper-personalized, data-backed insights, it acts as a virtual counselor that guides students seamlessly from their early high school choices all the way to specialized professional tracks.
